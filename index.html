Iniciativas

  • Meetups Mensuales

    Cada 2do jueves del mes tenemos una reunión virtual para reencontrarnos, aprender algo juntos o solo ponernos al día!

  • Club de Lectura

    Cada 2 semanas nos reunimos para discutir un nuevo capítulo del libro actual que estemos leyendo.

    Libro actual:
    Refactoring - Ruby Edition

  • Live Code Streaming

    Inactivo

    Cada 2 semanas, algunos organizadores hacemos un streaming en Twitch programando un proyecto en Ruby.

Todas nuestras actividades son gratuitas y esto es gracias al apoyo de la comunidad
